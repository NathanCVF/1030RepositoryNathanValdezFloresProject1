What Is the Stroop Test?

The Stroop Test is a simple task meant test for attentional issues. However, this version is not able to do what the average stroop test should do. It cannot loop and there is no button to activate it, the tab must be refreshed for a new set of word.

Stroop Test Checks:

  1. Process speed(Our reaction to conflicting stimuli and imagery)
  2. Cognitive interference(How well we can avoid auto reading and color identifying)
  3. The ability to filter out distractions

How to Use

  1. Click the 25 second count-down timer
  2. Identify the colored ink of words given
  3. Do not read the word itself
  4. Attempt to identify all words before the timer is up
  5. Refresh the tab for a new set of images if you want a new round
